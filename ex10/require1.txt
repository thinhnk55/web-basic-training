#Web Theme
Theme trong lập trình web là gì?

Less là gì?
Build file card.less thành card.css
Dùng card.css để sử dụng trong file card.html
Thử thay đổi các yếu tố trong card-new.less để biên dịch ra theme mới trong file card-new.css

Tìm hiểu sâu hơn về lập trình Less để có thể tùy biến theme theo ý thích ở đây:
https://lesscss.org/features/
