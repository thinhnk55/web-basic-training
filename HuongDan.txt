Cách làm:
1. Mở thư mục đọc file require.txt và mở thư mục demo xem ảnh demo.jpg để biết công việc mình cần phải thực hiện.
2. Trong tất cả các thư mục khác có tài nguyên cần thiết để làm. Nếu trường hợp không đủ tài nguyên thì liên hệ để hỏi.
3. Bài tập này yêu cầu thực hiện code trên 2 file là login.html và css/login.css/login
4. Công cụ lập trình khuyên dùng là Visual Code (nếu chưa biết sử dụng thì có thể liên hệ để hỏi). https://code.visualstudio.com/docs/editor/vscode-web

Yêu cầu khi làm bài:
Mục đích của bài tập là để đánh giá năng lực, do đó yêu cầu khi làm bài phải tự mình làm.
Không copy code của người khác trên mạng.
Làm được tới đâu thì thể hiện đến đó.
Qua đó người hướng dẫn sẽ có phương án đào tạo phù hợp.
Nghiêm cấm gian lận hoặc giấu dốt.
Trong quá trình làm bài khuyến khích đặt câu hỏi và thảo luận.
Tuy nhiên nghiêm cấm chia sẻ code với nhau.
Chỉ thảo luận về lý thuyết, nguyên lý, tư duy, cung cấp tài nguyên để học hỏi, chứ không code hộ.

Yêu cầu khi nộp bài:
1. Mỗi người phải tạo một tài khoản GitHub, và tạo một repository có tên là web-basic-training và để ở trạng thái private (để tránh copy bài của nhau)
2. Mời github của người hướng dẫn vào repo, tài khoản github là thinhnk55
2. Khi làm bài, mọi người tạo bên trong repo một thư mục con tương ứng với phần bài tập mình làm.
   Ví dụ trường hợp bài 1 thì tên thư mục là ex1 và để phần code của mình bên trong.
3. Khi nộp bài thì cần gửi lại link GitHub repository, người hướng dẫn sẽ tải về và chủ động đánh giá.
4. Nếu có thắc mắc hay câu hỏi gì thì vui lòng chủ động hỏi đáp. Đảm bảo đọc kĩ hiểu đúng mọi vấn đề trước khi làm.
